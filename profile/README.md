## Precise Vector Editing

The heart of <a href="#">download sketch for mac</a> is its vector engine. Every shape is editable,
scalable and exact, so designs stay crisp at any size and on any screen.

- **Boolean operations** — union, subtract, intersect and difference to build complex shapes from simple ones.
- **Editable paths** — adjust points, handles and curves directly on the canvas with full precision.
- **Rulers, guides and grids** — align and space elements exactly with pixel-accurate snapping.
- **Non-destructive scaling** — resize artwork and components without losing sharpness or detail.

Because <a href="#">vector design mac</a> keeps everything as live vectors, you are never locked into a
flattened raster. This precise, editable foundation is what makes Sketch dependable for interface work
where exact measurements and clean geometry matter.

## Symbols and Reusable Components

A defining feature of <a href="#">sketch for mac</a> is its component model. Symbols let you turn any
element — a button, a card, an icon — into a reusable piece that updates everywhere at once.

- **Symbols** — create a master element and reuse it across artboards and documents.
- **Overrides** — swap text, images and nested symbols per instance while keeping the master intact.
- **Smart Layout** — components resize and reflow automatically as their content changes.
- **Shared libraries** — publish components so a whole team designs from the same source of truth.

This makes <a href="#">sketch mac download</a> especially strong for maintaining consistency: change a
symbol once and every screen that uses it stays in sync, which is exactly what large interface projects
